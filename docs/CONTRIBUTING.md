# Contributing to XavaChain

We welcome contributions to the XavaChain core infrastructure!

## How to Contribute
1. **Report Bugs:** Open an issue regarding any $XAVA distribution or Gas Oracle discrepancies.
2. **Feature Requests:** Suggest new TMA (Telegram Mini-App) wallet features.
3. **Pull Requests:** - Fork the repo.
    - Create a feature branch.
    - Ensure all `L1GasPriceOracle` tests pass.
    - Submit a PR with detailed notes.

## Code of Conduct
Please be respectful to the community. Our goal is to build a sovereign L2 for everyone.
